# 部署 kafka

# 启动服务
```shell
docker-compose -f multi-zk-multi-kafka.yml up -d
```


### 停止服务
```shell
docker-compose -f multi-zk-multi-kafka.yml stop
```


### 停止并删除服务
```shell
docker-compose -f multi-zk-multi-kafka.yml down
```
