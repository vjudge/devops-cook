# cd
用于切换当前工作目录。


### 语法
```
cd [参数] [目录]
```

### 常用参数
* -P : 如果切换的目标目录是一个符号链接，则直接切换到符号链接指向的目标目录
* -L : 如果切换的目标目录是一个符号链接，则直接切换到符号链接名所在的目录
* -- ** 仅使用”-“选项时，当前目录将被切换到环境变量”OLDPWD”对应值的目录
* ~	切换至当前用户目录
* . : 则是表示目前所在的目录
* ..	切换至当前目录位置的上一级目录


### sample
```
# 跳转到/Users/vjudge/Downloads目录下（绝对路径）
cd /Users/vjudge/Downloads
# 跳转到home目录下
cd ~
# 跳转到当前目录的上上层目录下（相对路径）
cd ../../
```
